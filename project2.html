AsukaChat-1.3B 中文角色模型方案

面向 1.3B 参数中文角色聊天模型，设计从 Base Pretraining 到 Natural Chat SFT、Character SFT、 Character DPO 的完整阶段计划。当前项目处于设计冻结候选与数据源评审阶段，尚未下载数据或训练模型。

项目概述

AsukaChat-1.3B 是一个中文角色聊天模型项目设计。目标不是通用助手，而是完成一条角色一致性优先的训练链路： 先训练中文对话倾向的 base model，再通过自然聊天 SFT、角色 SFT 和 DPO 强化角色风格、非助手化表达和多轮稳定性。

方案要点

  • 模型目标：1.3B 参数、2048 context、纯中文聊天场景。
  • 训练路线：Base -> Chat-SFT -> Asuka-SFT -> Asuka-DPO。
  • 预训练配比：中文通用 20%、论坛/对话 35%、小说/对白 35%、情绪生活 5%、中文角色互动 5%。
  • Tokenizer 方向：SentencePiece BPE、50K vocab、特殊 token 与对话标记稳定性检查。
  • 评测核心：角色一致性、非助手行为、多轮稳定、中文自然度、非重复和不过度礼貌。

数据与安全边界

  • 候选数据源只作为 inventory 记录，未批准下载、抓取、清洗、tokenize 或训练。
  • 显式排除代码数据、工具调用、客服对话、法律/医疗建议、未经授权私聊和未审查版权文本。
  • 评测 prompt 与训练数据隔离，避免把回归评测样本混入训练集。
  • `.gitignore` 保护 raw / processed / tokenized 数据、模型权重、checkpoint、运行日志和密钥。
